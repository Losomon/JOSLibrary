# AI Library + Bookstore Backend

Next.js 15 App Router backend with Supabase, Grok AI, M-Pesa/Paystack payments.

## Structure
- `app/api/`: API routes
- `app/actions/`: Server Actions
- `app/lib/`: Utilities & services
- `app/middleware.ts`: Auth middleware

## Setup
1. Update .env.local with keys
2. `npm install`
3. `npm run dev`
